# v0.2.0 — Feature: Smart Filename & UI Fix

**Released:** 2026-05-07

---

## New Features

### Downloader

- **[Meaningful filename for fallback downloads]**: When yt-dlp cannot handle
  a URL directly and falls back to the page scraper, the downloaded file is
  now named using the page <title> instead of a random UUID from the video URL

## Bug Fixes

### GUI

- **[Disabled window resize and maximize]**: The application window is now
  fixed-size to prevent layout issues when maximized

## CI and Tests

### Test Runner

- **[Fixed pytest exit code 5 under bash -e]**: CI no longer fails when
  pytest exits with code 5 (no tests collected) due to the bash -e flag

## Versioning

> `Minor` bump: new filename feature added.
> Bug fix and CI improvement included.
